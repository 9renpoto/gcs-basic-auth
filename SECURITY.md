# Security Policy

## Supported Versions

プログラムの分量は少ないので、セキュリティ問題があってもライブラリで発生すると思います。
npmインストール時にライブラリが古いなどの警告が出た場合は、内容にしたがって修正してください。

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report in issue.
(issueで報告頂ければ対応します。)
